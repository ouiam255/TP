# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-22

### Added
- Initial release of Spring Boot Kubernetes Demo
- REST API with greeting endpoint (`/api/hello`)
- Kubernetes ConfigMap integration for externalized configuration
- Spring Security with CORS support
- MySQL database integration with JPA/Hibernate
- Docker containerization with optimized Alpine-based image
- Kubernetes deployment manifests (Deployment, Service, ConfigMap, Namespace)
- Health check probes (liveness and readiness)
- Resource limits and requests configuration
- Comprehensive JavaDoc documentation
- Professional README with setup instructions
- Automated deployment and cleanup scripts
- Docker Compose for local development
- MIT License
- Contributing guidelines

### Technical Details
- Java 17
- Spring Boot 3.2.0
- Maven build system
- MySQL 8.0 compatible
- Kubernetes compatible
- Docker multi-stage build ready

### Documentation
- Complete API documentation
- Architecture diagrams
- Deployment guides
- Troubleshooting section
- Configuration reference
